**Utilisation :**

se placer dans src
compiler les sources java
puis taper la commande suivante

./futoshiki ../tests/fichier_tests.FUTO

Remplacer fichier_tests.FUTO par un fichier au format FUTO

placez le bien dans le repertoire tests.
